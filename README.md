# Git Practice Repo

## Mục tiêu
Ôn lại toàn bộ Git từ cơ bản → nâng cao theo dạng bài tập thực hành.

---

## Level 1: Basic
- git init
- git add / commit
- git log
- git diff

## Level 2: Branching
- git branch
- git checkout / switch
- git merge

## Level 3: Rebase & Conflict
- git rebase
- xử lý conflict

## Level 4: Undo
- git reset
- git revert
- git stash

## Level 5: Remote
- git push / pull
- git fetch
- xử lý conflict khi pull

---

## Rule học
- Mỗi bài = 1 commit
- Không copy solution
- Khi bí → dùng `git log`, `git status`
